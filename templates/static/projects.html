Progressive Inner Source Bug Analysis Excel Macro

Description:

During my first internship my manager had came to me with a side project to keep me busy in any down time I may have. The motivation behind the project came from her spending hours importing and manipulating data regarding to bugs in Azure Dev Ops. She wanted me to see if I was able to automate this process to save her time on her quarterly reports. I ended up being able to do exactly that which saved a few hours of business time each quarter. Furthermore, I presented and demoed the side project to multiple group managers and directors withing the IT department!

Since this project was done internally for my manager using Progressive Data it is obviously not publicly available, with that said, the tech stack consisted of using Virtual Basic for Applications within Microsoft Excel and creating an easily distributable batch script that installs itself to the Excel ribbon automatically using the user variable. The application then ran on top of the users excel window providing a user interface for filtering and creating dynamic analysis graphs of the selected data.
